Basic HTML and HTML5

Say Hello to HTML Elements

Скажите привет HTML элементам

  • Welcome to freeCodeCamp's HTML coding challenges. These will walk you through web development step-by-step.

    Добр пожаловать в проблемы с кодированием HTML-кода freeCodeCamp. Постепенно вы будете проходить через веб-разработку.

  • First, you'll start by building a simple web page using HTML. You can edit code in your code editor, which is embedded into this web page.

    Во-первых, вы начнете с создания простой веб-страницы с использованием HTML. Вы можете отредактировать code в редакторе кода, который встроен в эту веб-страницу.

  • Do you see the code in your code editor that says <h1>Hello</h1>? That's an HTML element.

  • Most HTML elements have an opening tag and a closing tag.

  • Opening tags look like this:

  • <h1>

    <h1>

  • Closing tags look like this:

  • </h1>

  • The only difference between opening and closing tags is the forward slash after the opening bracket of a closing tag.

  • Each challenge has tests you can run at any time by clicking the "Run tests" button. When you pass all tests, you'll be prompted to submit your solution and go to the next coding challenge.

  • To pass the test on this challenge, change your h1 element's text to say "Hello World"
